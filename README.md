# model_cool 模酷酷

## 项目是一个模型爬虫加模型后台管理系统项目

#### 内部采用java编程语言,使用了Java爬虫框架WebMagic,虚拟内核浏览器模拟,幻影无头浏览器,调用谷歌浏览器模拟

#### 具体实现可以参考:model_biz_reptile工程源代码

---
